# start-html-template
Start HTML Template.

Guidelines: https://github.com/chris-pearce/css-guidelines

# How start
1. git clone https://github.com/Shmasya/start-html-template/
2. cd start-html-template
3. npm run init
4. npm run serve
# For build to Prod
1. npm run build
# For zip src only
1. npm run zip
